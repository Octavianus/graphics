### Additional Features:
- implemented the cube rotation around its diagonal in both mode.
- implemented the lighting effect in both mode.

### Operations:
 *      "b" switch to basic mode.
 *      "g" switch to GL mode.
 *      "q" quit the program.
 *      "s" in GL mode, stop the rotation.
 *      "r" in GL mode, restart the rotation.
